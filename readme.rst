###################
Kelas Yuk!
###################

Kelas Yuk merupakan aplikasi kelas virtual yang dapat mempertemukan pengajar dengan murid
dengan tagline dimana saja, kapan saja, siapa saja. Kelas Yuk dibuat dalam rangka membuat
proses belajar tidak perlu dilakukan secara tatap muka namun dapat dilakukan dimana saja
dan menjangkau banyak murid di seluruh dunia.
